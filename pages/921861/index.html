# 代码托管

GitHub

github是一款分布式的版本控制系统，是一个面向开源及私有软件项目的托管平台。

码云

码云是开源中国社区团队推出的基于Git的快速的、免费的、稳定的在线代码托管平台。

Bitbucket

BitBucket 是一家源代码托管网站，采用Mercurial和Git作为分布式版本控制系统。

Gitlab

GitLab 是一个基于 git 的仓库管理程序,也是一个方便软件开发的强大完整应用。
